# JobTrackr Client

React + TypeScript + Vite frontend for JobTrackr application.

## Running

```bash
npm run dev
```

Frontend will start on http://localhost:5173

## Features

- Authentication (Login/Register)
- Dashboard with statistics
- Application management (CRUD)
- Search and filtering
- Admin panel
- Protected routes
